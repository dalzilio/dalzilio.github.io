We have recently started work on a new model-checking tool called PåNĐA, for
time Petri Nets Diagnosability Analyzer.

PaNDA is a formal verification tool for the _state estimation_, _prognosis_ and
_diagnosability_ of time Petri nets (TPN). It has been built to support a
diagnoser-based approach and complements our previous work with the tool
[Twina](https://projects.laas.fr/twina/), which implements a twin-plant
approach.
