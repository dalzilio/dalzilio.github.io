Just like "data is the new oil" (a valuable resource that must be refined and
processed to be useful), models and benchmark data are the lifeblood of software
competitions.

I have recently made available a Gitlab repository containing a collection of
models that I have submitted as benchmark for the [Model-Checking
Contest](https://mcc.lip6.fr/models.php) along the years.

While the MCC website features a selection of model instances, the models
presented here are parameterized and include scripts or programs for generating
arbitrary new instances, as needed. This resource is intended to facilitate the
sharing of new models ; to provide easy access to the accompanying scripts ; but
also to serve as a reference for people looking to contribute their own models.
In particular, the code contains examples of the LaTeX description files
included with each model submission.
